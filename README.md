# Iron Progress 💪

Фитнес-трекер для отслеживания тренировок, прогресса и достижений.

🔗 **[Демо версия](https://akhmed265.github.io/iron-progress/)**

## 🎯 Основные возможности

- 📅 Планировщик тренировок с календарем
- 💪 Библиотека упражнений (50+ упражнений)
- 📊 Статистика и визуализация прогресса
- ⏱️ Таймер тренировок
- 📝 Дневник тренировок
- 🎯 Система целей и достижений
- 🎨 Анимированный логотип-гантель
- 🌙 Темная/светлая тема

## 🚀 Технологии

Vue 3 • TypeScript • SCSS • Vite • Pinia • Vue Router • Day.js

## 📦 Установка

```bash
npm install
npm run dev
```

## 💾 Хранение данных

Все данные хранятся в LocalStorage браузера.
